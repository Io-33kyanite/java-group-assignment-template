# java-assignment-template

How to use this template

1. このリポジトリをForkしてください。
2. settings.gradleのproject名を変更してください。
3. README.mdに課題の説明を記載してください。
4. src/test/java配下にテストコードを記載してください。
5. gradle wrapperを使ってビルド・実行してください。
   - ローカルでのテスト実行: `./gradlew test`
   - メインクラスの実行: `./gradlew run`
6. `set the assignment`で課題用リポジトリの準備完了です。
